# RealTimeCamera
應用於Android上的即時影像辨識範例，採用Tensorflow Lite函式庫，能夠運行在搭載鏡頭的Android裝置與ASUS Zenbo與Zenbo Junior機器人上。 

1. ObjectDetection
2. GunBinaryClassification
3. LogoClassification
